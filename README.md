## 🎯 What is Prio?

Prio is a meticulously crafted priority management application built from the ground up. Designed for those who value their time and need a distraction-free way to manage what truly matters.

The name itself reflects its core philosophy—**Prio** stems from "Priority," embodying a singular focus: helping you identify and tackle what's important, not just what's urgent.

---

## 💡 The Story Behind Prio

This app emerged from a simple observation: existing priority management tools often overcomplicate what should be simple. What if we could strip away unnecessary features and create something that *just works*?

Built entirely from scratch over an intensive **20-day development cycle**, Prio represents countless hours of iteration, testing, and refinement. Every feature was thoughtfully considered. Every interaction was optimized. Every pixel was intentional.

**No borrowed designs. No templates. No shortcuts.**

Just pure, deliberate problem-solving with **13+ hours of daily commitment**.

## 🔧 How to Install

### **Step 1: Enable Installation from Unknown Sources**
```
Settings → Security/Privacy → Unknown Sources → Toggle ON ✅
(Path varies by device manufacturer)
```

### **Step 2: Download Prio APK**
```
Click the DOWNLOAD button above
→ GitHub releases page opens
→ Download "app-debug.apk" file
```

### **Step 3: Install the App**
```
1. Open File Manager / Downloads
2. Find "app-debug.apk" 
3. Tap the file
4. Tap "INSTALL"
5. Wait 10-30 seconds
6. Tap "OPEN" to launch 🎉
```

## 🐛 Troubleshooting

### "App not installed" Error
```
✓ Check Android version (5.0+)
✓ Enable "Unknown Sources"
✓ Ensure 50MB+ free storage
✓ Delete incomplete downloads
✓ Try again
```

### App crashes on startup
```
✓ Uninstall completely
✓ Restart your phone
✓ Download fresh from releases
✓ Reinstall
✓ Report if issue persists
```

### App is slow
```
✓ Close background apps
✓ Clear app cache (Settings → Apps → Prio → Clear Cache)
✓ Restart device
✓ Update to latest version
```
